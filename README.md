# Credit Card Fraud Detection Analysis

This project analyses credit card transaction data to identify fraudulent patterns and suspicious behaviour using Python, Tableau, and Excel.

## Dataset
Kaggle Fraud Test Dataset  
Size: 555,657

## Tools Used
- Python (Pandas, Data Cleaning, Analysis)
- Excel (Data Validation & Exploration)
- Tableau (Data Visualisation & Dashboard Creation)

## Project Overview
- Cleaned and preprocessed transaction data using Python  
- Performed exploratory data analysis to identify fraud patterns  
- Developed a rule-based fraud detection approach using transaction thresholds  
- Used Excel for validation and supporting analysis  
- Built interactive Tableau dashboards to visualise fraud trends and anomalies  

## Project Links
- Kaggle Notebook/Dataset: https://www.kaggle.com/datasets/chetanmittal033/credit-card-fraud-data

## Key Insights
- Identified unusual transaction behaviours linked to fraud  
- Highlighted patterns in transaction frequency and amounts  
- Provided visual insights for improved fraud detection decisions
